# 🚀 Quick Start: Upload Manual Kontak\n\n## Dalam 3 Menit!\n\n### Step 1: Buka Menu\n→ Pergi ke: **http://localhost/email_dispatcher_suite/public/contacts.php**\n\n### Step 2: Klik Button Upload\nCari bagian **\"Sinkronisasi Outlook\"** dan klik tombol **\"📤 Upload Manual Kontak\"\n\n### Step 3: Pilih Opsi\nDialog akan muncul dengan 2 pilihan:\n\n**Option A: Belum Punya Template?**\n- Klik **\"⬇️ Download Template\"**\n- File CSV akan didownload dengan nama: `contact_template_id_[NUMBER].csv`\n- Buka di Excel/Google Sheets\n- Isi data kontak Anda (hapus contoh rows yg tidak perlu)\n- Save file\n- Ulangi Step 2, lalu pilih \"📁 Sudah Ada, Upload File\"\n\n**Option B: Sudah Punya File CSV?**\n- Klik **\"📁 Sudah Ada, Upload File\"**\n- Lanjut ke Step 4\n\n### Step 4: Upload File\nGunakan salah satu cara:\n\n**Cara 1: Drag & Drop**\n- Drag file CSV ke area yang bertuliskan \"📄 Drag & drop file CSV di sini\"\n- File otomatis terupload\n\n**Cara 2: Browse File**\n- Klik pada area upload\n- Pilih file CSV dari komputer Anda\n- Tekan Open\n\n### Step 5: Lihat Hasil\nSetelah upload selesai, akan muncul ringkasan:\n```\n✅ Upload Berhasil!\n\nDiproses:        25 baris\nDitambahkan:     23 kontak ✓\nSkip (duplicate): 2 email ⚠️\nSkip (kosong):    0 baris\nError:            0 item\n```\n\n### Step 6: Selesai\nPilih opsi:\n- **\"Unggah File Lain\"** → Upload file lain\n- **\"Selesai & Refresh\"** → Refresh halaman & lihat kontak baru\n\n---\n\n## 📋 Format CSV Yang Diminta\n\n### Minimal Format\n```\nEmail\ncontact1@example.com\ncontact2@example.com\n```\n\n### Recommended Format\n```\nName,Email\nJohn Doe,john@example.com\nJane Smith,jane@example.com\nCompany Name,info@company.com\n```\n\n### Tips\n✅ Kolom \"Email\" WAJIB ada\n✅ Kolom \"Name\" optional (jika tidak ada, email digunakan sebagai nama)\n✅ Ubah nama kolom sesuka hati: \"Email\", \"email\", \"EMAIL\", \"Email Address\" - semuanya diterima\n✅ Maksimal 10MB per file\n✅ Format: CSV (comma-separated) bukan XLSX/Excel\n\n---\n\n## 🎯 Apa Yang Terjadi?\n\n### Data Existing\n- ✅ Tetap aman (tidak akan dihapus)\n- ✅ Email duplikat akan skip (tidak diupdate)\n\n### Data Baru\n- ✅ Ditambahkan ke database\n- ✅ Source ditandai sebagai \"Manual Upload\"\n- ✅ Auto timestamp untuk last_synced\n\n### Contoh Skenario\nJika Anda upload 10 email dimana 2 email sudah ada:\n- 8 email baru → ditambahkan\n- 2 email lama → skip (tidak diupdate, data tetap original)\n\n---\n\n## ❓ FAQ\n\n**Q: Apakah data existing akan terhapus?**\nA: Tidak! Hanya INSERT (tambah), tidak ada DELETE atau UPDATE.\n\n**Q: Bagaimana jika email sudah ada?**\nA: Email duplikat akan skip otomatis. Data existing tidak berubah.\n\n**Q: Berapa maksimal file yang bisa diupload?**\nA: 10MB (~10,000+ baris tergantung ukuran data)\n\n**Q: Apakah bisa upload berkali-kali?**\nA: Ya! Email duplikat akan otomatis skip.\n\n**Q: Format apa yang didukung?**\nA: CSV only (`.csv`). Jika punya `.xlsx`, save as CSV dulu di Excel.\n\n**Q: Bagaimana caranya lihat kontak yang sudah diupload?**\nA: Scroll ke bawah di halaman yang sama, ada section \"Daftar Kontak\".\n\n---\n\n## 🆘 Troubleshooting\n\n### \"File harus berformat CSV\"\n→ Pastikan file extension adalah `.csv`, bukan `.xlsx` atau lainnya\n\n### \"CSV harus memiliki kolom Email\"\n→ Tambahkan kolom \"Email\" di header CSV Anda\n\n### \"File terlalu besar (maks 10MB)\"\n→ Pisahkan file menjadi beberapa bagian, upload satu-satu\n\n### Kontak tidak muncul setelah upload\n→ Refresh halaman (F5 atau Ctrl+Shift+R)\n→ Check di summary apakah ada \"Skip (duplicate)\"\n→ Lihat di section \"Daftar Kontak\"\n\n---\n\n## ✨ Tips & Tricks\n\n💡 **Gunakan template yang didownload** untuk memastikan format konsisten\n\n💡 **Batch multiple files** - tak perlu worry duplikasi, akan auto skip\n\n💡 **Add custom column** - bisa tambah kolom lain (akan diabaikan)\n\n💡 **Excel tips for save as CSV**:\n   - File → Save As\n   - Format: \"CSV (Comma delimited)\" atau \"CSV UTF-8\"\n   - Simpan\n\n---\n\n## 🎉 Selesai!\nKontak Anda sekarang sudah tersimpan dan siap digunakan untuk:\n- 📧 Compose email\n- 👥 Grup kontak\n- 📤 Bulk send\n\nHappy emailing! 🚀\n
